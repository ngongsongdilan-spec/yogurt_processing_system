System Modules

Every aspect of yoghurt production, from raw milk intake to final sale, managed in one place.

📊

Dashboard

Real-time overview of active batches, stock levels, QC failures, machine status, pending orders, and recent changes.

📦

Inventory

Track raw materials and finished goods with lot-level traceability, expiry dates, and reserved quantities.

🏭

Production Batches

Plan and monitor batch production from recipe assignment to completion, with full status tracking.

🔬

Quality Control

Record inspection results per batch with pass/fail/hold status. Completed QC records are locked against modification.

⚙️

Machines

Manage equipment inventory, operational status, and preventive maintenance schedules with cost tracking.

📋

Audit Trail

Automatic change tracking via PostgreSQL triggers — every INSERT, UPDATE, DELETE is logged with who and when.

🛒

Purchasing & Sales

End-to-end purchase orders with suppliers and sales orders to customers, including line-item details and totals.

🔐

Security Sandbox

Interactive tests for PostgreSQL triggers: completed batch lock, QC approval lock, check constraints, and unique violations.

💾

SQL Explorer

Run read-only SQL queries directly against the database to inspect tables, joins, and raw data.

User Roles & Permissions

Each role is tailored to specific responsibilities within the yoghurt processing workflow.

A

Admin

Full system access — manage users, oversee all modules, configure roles, and access sensitive audit data.

M

Manager

Production oversight — create and approve batches, review inventory, analyze QC results, and generate reports.

O

Operator

Daily operations — start and complete production batches, record material consumption, and update machine status.

I

Inspector

Quality assurance — perform inspections, record results, and flag batches for review. Passed inspections are locked.

V

Viewer

Read-only access — view dashboards, browse reports, and inspect audit logs without making any changes.
